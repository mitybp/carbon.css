Flex container

<element class="d-flex"></element>

Adds flex property to element.


<element class="d-flex flex-[direction]-[optional: reverse]"></element>

Adds flex container direction, flex-row is default. See options below.

  • flex-row
  • flex-row-reverse
  • flex-column
  • flex-column-reverse

<element class="d-flex flex-[wrap or nowrap]-[optional: reverse]"></element>

Adds the breaking of items inside the flex container. Can be used independently of other customizations.

  • flex-wrap
  • flex-wrap-reverse
  • flex-nowrap

<element class="d-flex [justify or align-items]-[options]"></element>

Sets the vertical (align-items) and horizontal (justify) position of the items within the flex container.

Align items:

  • align-items-center
  • align-items-start
  • align-items-end

Justify content:

  • justify-center
  • justify-start
  • justify-end
  • justify-evenly
  • justify-between
  • justify-around
